# Tripleten web_project_around
El nombre del proyecto

AROUND THE US.

Una descripción del proyecto y su funcionalidad

Perfil de usuario para una web de viajes por los Estados Unidos en ella se edita el perfil, se puede agregar nuevas imagenes, gestionar la interaccion como "me gusta" y eliminacion de imagenes. 


Una descripción de las tecnologías y técnicas utilizadas:
HTML5
CSS
Diseño responsivo
JAVASCRIPT(En este proyecto utilizamos clases de JavaScript.)
manejo de API REST, Fetch y validación de formularios.

Logros destacables:
Implementé una arquitectura escalable con clases reutilizables, integración completa con backend y manejo de estados de interfaz, fortaleciendo mis habilidades en JavaScript y trabajo con APIs reales.
